# AI Players 2026

> **AI를 실질적으로 주도하는 기업 10선과 그들의 제품·프로그램을 정리한 인터랙티브 레퍼런스 페이지**

🔗 **Live:** [rookiejj.github.io/ai-players](https://rookiejj.github.io/ai-players)

---

## 왜 만들었나

AI 관련 발표와 제품 출시가 하루가 멀다 하고 쏟아지는 지금, 정작 "지금 이 시점에 어떤 기업이 무엇을 갖고 있는가"를 한눈에 정리한 레퍼런스가 없다는 문제의식에서 출발했다.

뉴스 기사는 단편적이고, 위키피디아는 느리게 업데이트되며, 각 기업의 공식 페이지는 자사 제품만 다룬다. **AI Players**는 그 사이의 공백을 메우기 위한 큐레이션된 스냅샷이다.

단순한 모델 성능 비교표가 아니라, **전략적 맥락** — 왜 이 기업이 이 제품을 만들었는지, 시장에서 어떤 포지션을 가져가는지 — 를 함께 기록하는 것을 목적으로 한다.

---

## 선정 기준

"빅테크"라는 기준 대신, **AI 생태계를 실질적으로 형성하는 플레이어**를 우선순위로 선정했다.

| 기준 | 설명 |
|---|---|
| **프론티어 모델 개발** | 자체 LLM·멀티모달 모델을 직접 연구·출시하는가 |
| **생태계 영향력** | 다른 기업/개발자들이 이 기업의 플랫폼·표준 위에서 개발하는가 |
| **인프라 지배력** | AI 전체 산업이 이 기업의 하드웨어·클라우드에 의존하는가 |
| **오픈소스 영향** | 공개 모델·도구가 업계 표준으로 자리잡았는가 |
| **시장 규모 및 속도** | 실사용자 수, 매출, 성장 속도 |

이 기준에 따라 순수 매출 기준 빅테크(IBM, Oracle 등)보다 **Anthropic, xAI, Mistral AI** 같은 AI 전문 기업이 우선 포함된다.

## 구성 및 설계 원칙

### 수록 정보
각 기업별로 다음을 기록한다.

- **핵심 AI 전략** — 기업의 전체 방향성 한 줄 요약
- **제품·프로그램 목록** — 현재 활성 상태인 주요 AI 제품/서비스
- **제품별 설명** — 기술 스펙 + 시장 맥락 (단순 스펙 나열 지양)
- **카테고리 배지** — LLM / 이미지 / 비디오 / 에이전트 / 코딩 / UI 디자인 / 하드웨어 / 과학AI
- **NEW 표시** — 최근 1개월 이내 출시·업데이트된 항목
- **공식 링크** — 기업명 및 제품명 클릭 시 해당 공식 페이지로 직접 이동

### UI 설계
- 카테고리 필터로 원하는 분야만 빠르게 탐색 가능
- 다크 테마, 반응형 (모바일 대응)
- 단일 HTML 파일 — 외부 의존성 최소화 (Google Fonts 제외)

## 업데이트 방법

이 페이지는 **정기적인 업데이트**를 전제로 설계됐다. AI 분야 특성상 분기 단위로도 판도가 바뀌기 때문에 최소 월 1회 리뷰를 권장한다.

### 새 제품/항목 추가

`index.html` 내 JavaScript `data` 배열에서 해당 기업 객체를 찾아 `products` 배열에 항목을 추가한다.

```js
{
  name: "제품명",           // 표시될 제품명
  url: "https://...",      // 클릭 시 이동할 공식 URL
  isNew: true,             // 최근 출시면 true → NEW 배지 표시
  badges: ["llm", "agt"],  // 카테고리 (아래 목록 참고)
  desc: "설명 텍스트"       // 기술 스펙 + 시장 맥락 포함 권장
}
```

**사용 가능한 badges 값:**

| 값 | 의미 |
|---|---|
| `llm` | 언어 모델 |
| `img` | 이미지 생성 |
| `vid` | 비디오 생성 |
| `agt` | 자율 에이전트 |
| `code` | 코딩 특화 |
| `ui` | UI/디자인 AI |
| `hw` | AI 칩·인프라 |
| `sci` | 과학·연구 AI |

### 새 카테고리 추가

1. CSS `:root`에 색상 변수 추가:
   ```css
   --b-새카테고리: #배경색;
   --t-새카테고리: #텍스트색;
   ```
2. CSS에 `.b-새카테고리` 클래스 추가
3. `BL` 객체에 `새카테고리: "표시명"` 추가
4. toolbar `<nav>`에 필터 버튼 추가
5. legend에 항목 추가

### 기업 추가/교체

`data` 배열에 아래 형식으로 객체를 추가한다.

```js
{
  rank: 11,
  name: "기업명",
  url: "https://...",
  type: "기업 유형",
  focus: "핵심 AI 전략 한 줄",
  products: [ /* 위 형식 */ ]
}
```

### NEW 배지 관리

출시 후 약 4~6주가 지난 항목은 `isNew: false`로 변경해 배지를 제거한다.
